*Interactive Tableau dashboard analyzing pricing distribution, demand indicators, and spatial patterns of Airbnb listings in New York City.*

---

## Project Objective
The goal of this analysis is to:
- Understand how prices are distributed across different room types
- Examine the relationship between price and demand (number of reviews)
- Identify spatial patterns of listings across NYC boroughs

---

## Dashboard Overview
The interactive dashboard includes:
- Price distribution analysis
- Price comparison by room type
- Price vs number of reviews
- Availability vs price
- Geographic distribution of listings (map)
- Room type distribution and top neighbourhoods by average price

---

## Tools & Technologies
- Tableau Public
- CSV dataset (Airbnb NYC Open Dataset)
- Exploratory Data Analysis (EDA)

---

## Tableau Analysis Details
- Applied price filtering to handle extreme outliers and improve visual interpretability
- Used distribution plots (histogram, boxplot) to analyze skewed price data
- Explored relationships between price and demand using scatter plots
- Built an interactive dashboard with global filters for room type and area
- Focused on clarity, consistency, and analytical storytelling rather than visual overload

---

## Calculated Fields
- Created simple calculated fields to support exploratory analysis (e.g. price-based comparisons and availability indicators).

## Key Insight
Lower-priced listings tend to receive more reviews, while higher-priced listings are more concentrated in central areas such as Manhattan.
